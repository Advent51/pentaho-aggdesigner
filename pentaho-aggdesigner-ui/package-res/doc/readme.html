Pentaho® Aggregation Designer, Release 1.0 M1

Introduction

Thank you for choosing the Pentaho Aggregation Designer. The Pentaho Aggregation Designer provides you with a simple interface that allows you to create aggregate tables from the levels and dimensions you specify. Based on these selections, the Aggregation Designer generates the Data Definition Language (DDL) for creating the aggregate tables, the Data Manipulation Language (DML) for populating them, and an updated Mondrian schema which references the new aggregate tables. For users unfamiliar with aggregate table design concepts, the Aggregation Designer also includes an intelligent advisor that evaluates the structure and cardinality of their OLAP cube and recommends some initial aggregate tables to create for improving performance.

Important: Plug-ins for Teradata® AJI support are not included in the open version of the Pentaho Aggregation Designer.

Special Considerations

  • The Pentaho Aggregation Designer and all tools require JVM 1.5; JVM 1.6 is not supported at this time.

  • To connect to your specific database, you must provide a JDBC Driver for your database in the Drivers directory of the Pentaho Aggregation Designer.

  • To deploy and use Mondrian Aggregate Tables successfully, you must enable aggregate table support on your Mondrian server.
    For more information on aggregate table concepts and how to enable them in Mondrian, see Properties that Affect Aggregates.

Installation Instructions

The pad-open-1.0-xx.zip file contains all the libraries and script files necessary to run Pentaho Aggregation Designer. To install the Pentaho Aggregation Designer, unzip this file into a directory of your choice.

To launch the Aggregation Designer on Windows...

Run the startaggregationdesigner.bat script located in the root of your installation directory.

To launch the Aggregation Designer on Linux...

Run the startaggregationdesigner.sh script located in the root of your installation directory.

Important: Place your JDBC driver JARs in the Drivers directory. Once in this directory, the drivers are added to the classpath automatically when the Pentaho Aggregation Designer starts.

Known Issues in this Release

The following is a list of known issues in this release. Read it carefully before installing the product:

  • The use of Java 1.6 can cause performance issues when launching the open dialog. This is a known issue in Java 1.6. If you are experiencing this problem, install Java 1.5 or explicitly set your PATH to use a 1.5 JRE.

  • The Aggregate Advisor does not currently consider existing aggregations when making recommendations.
